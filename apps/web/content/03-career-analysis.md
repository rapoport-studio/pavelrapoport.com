## Ранний период (2003–2011)

### MidiCom Technology (2003–2006)
**Домен:** E-commerce / Hardware Distribution
**Что делал:** Разработка и поддержка e-commerce сайта дистрибьютора Asus и ZALMAN. Email-маркетинг.
**Контекст:** Первая профессиональная работа. Израиль, несколько лет после иммиграции.

### WebStudio (2008–2009)
**Домен:** Media / Entertainment / CMS
**Что делал:** BabyTv.com — 40+ модулей, CMS, мультиязычный контент. Фронтенд-разработка.
**Что вынес:** Масштабный CMS-проект. Первый опыт работы с многоязычным контентом — навык, который потом проявится в каждом проекте.

### OnElement (2009–2010)
**Домен:** E-commerce / Fashion Tech / 3D
**Что делал:** Flash (ActionScript 3), 3D-модели для онлайн-конфигуратора столов для мероприятий. Дизайн + разработка.
**Что вынес:** Визуальное программирование. Рендеринг сложных объектов в браузере. Первый контакт с тем, что позже станет главной специализацией — визуализация данных.

### Interminds (2010–2011)
**Домен:** E-commerce / Marketplace
**Что делал:** Reverse auction платформа Bidsarena.com. SQL Server, ADO.NET, Entity Framework, REST APIs, JSON. Full-stack: БД + API + фронтенд + CMS.
**Что вынес:** Полный цикл: от базы данных до UI. Понимание marketplace-модели.

## Стартапы и рост (2011–2026)

### 1. Soluto (2011–2013) → приобретена Asurion за ~$100–130M
**Домен:** Consumer Tech / Device Management / Remote Support
**Что делала:** Облачная платформа для удалённого управления и диагностики ПК. Помогала обычным людям решать технические проблемы на своих устройствах. Победитель TechCrunch Disrupt 2010. 3M+ скачиваний.
**Что я строил:** Веб-портал для удалённого управления устройствами. Визуализация состояния ПК, remote actions, real-time диагностика.
**Что я вынес:** Первый опыт SaaS-продукта с массовой аудиторией. Понимание, как сделать сложное техническое действие простым для обычного пользователя.

---

### 2. Wibbitz (2012–~2015) → приобретена Vimeo (2021)
**Домен:** Media Tech / AI Video / Content Automation
**Что делала:** AI-платформа для автоматического создания видео из текста. Запатентованная text-to-video технология. Клиенты: Bloomberg, Reuters, HubSpot, Condé Nast. Награды: Grand Prix UNESCO, Best Mobile Publishing Product (MWC Barcelona).
**Что я строил:** Веб-плеер и сайт. Canvas-based рендеринг видео, визуальный интерфейс редактора. Работа с медиа-контентом в реальном времени.
**Что я вынес:** Работа с Canvas API, рендеринг медиа в браузере. Первый контакт с AI (NLP для обработки текста в видео). Понимание медиа-индустрии изнутри.

---

### 3. Xplenty (2014–2018) → переименована в Integrate.io (слияние 4 компаний)
**Домен:** Big Data / ETL / Data Integration
**Роль:** Front-End Team Leader
**Что делала:** Low-code ETL-платформа. Drag-and-drop интерфейс для построения data pipelines: извлечение данных из десятков источников (Salesforce, MongoDB, PostgreSQL, S3), трансформация, загрузка в data warehouses (Redshift, BigQuery, Snowflake). Xplenty была позже объединена с DreamFactory, FlyData и Intermix.io в единую платформу Integrate.io.
**Что я строил:** SVG-based визуальный редактор data pipelines — core product feature. Миграция Backbone.js → Angular. Руководил фронтенд-командой.
**Что я вынес:** Глубокое понимание data flows и ETL-процессов. Опыт построения сложных визуальных редакторов (SVG node-graph). Первый опыт технического лидерства команды.

---

### 4. Perimeter 81 (2018–2019) → приобретена Check Point за $490M
**Домен:** Cybersecurity / SASE / Zero Trust Network Access
**Что делала:** Cloud VPN и Security Service Edge (SSE) платформа. Zero Trust Network Access для удалённых сотрудников. 3,000+ клиентов, 200+ сотрудников. Была оценена в $1B (Series C, 2022), продана за $490M.
**Что я строил:** Design system с нуля за один год — для 3 платформ (Web App, CRM, Landing Portals). Проектировал security entities: network connections, device management, access policies. Синхронизация Figma → Storybook. Нанял и обучил фронтенд-команду из стажёров.
**Что я вынес:** Первый полный design system от нуля до production. Понимание cybersecurity-домена изнутри. Опыт построения команды с нуля. Стартап-культура на ранней стадии.

---

### 5. brand DNA (2019) — собственный стартап
**Домен:** Developer Tools / Design Systems
**Роль:** Founder, VP R&D
**Что делал:** Theme manager для цифровых брендов — пространство для создания и поддержки общего визуального языка между дизайнерами и разработчиками. Инфраструктура, RESTful API, команда из 4 человек.
**Почему важно:** Это не случайный стартап. Это попытка продуктизировать свою главную суперсилу — design systems. Я увидел проблему, которую решал вручную в каждой компании, и попытался сделать из неё продукт.

### 6. QuickWork (2019–2020) — co-founder
**Домен:** Marketplace / HR Tech / Blue-collar
**Роль:** Co-Founder, VP R&D
**Что делал:** Платформа найма для рабочих профессий (уборка, переезды, гостиничный бизнес). React + Node.js, real-time matching, scheduling, payments. Презентации инвесторам, которые обеспечили финансирование.
**Что вынес:** Full product ownership: от идеи до инвестиционного питча. Понимание marketplace-бизнес-модели. Управление кросс-функциональной командой.

---

### 7. Zerto (2020–2022) → приобретена HPE за $374M
**Домен:** Enterprise / Disaster Recovery / Cloud Data Protection
**Что делала:** Лидер рынка continuous data protection (CDP). Журнальная технология для мгновенного восстановления данных после ransomware-атак, катастроф, сбоев. 9,000+ клиентов, 350+ MSP-партнёров. Основана в 2009, приобретена HPE для интеграции в GreenLake.
**Что я строил:** Миграция AngularJS 1.5 → React. Монорепо: 38,490 коммитов, 215 контрибьюторов. Design system @zerto/shared-fe: 56+ компонентов, Atomic Design, dual-format npm package. D3.js топология-графы, heatmaps, sparklines, real-time RPO мониторинг.
**Что я вынес:** Enterprise-масштаб. Работа с легаси-кодом и поэтапная миграция. D3.js и сложная визуализация данных. AG Grid Enterprise. Менторство команды, код-ревью культура.

---

### 8. HPE — Hewlett Packard Enterprise (2022–present)
**Домен:** Enterprise Cloud / Storage / GreenLake Platform
**Что делала:** Глобальная IT-инфраструктурная компания. GreenLake — edge-to-cloud платформа для enterprise-клиентов. Zerto стала частью HPE Storage.
**Что я строил:** Lead contributor (1,498 коммитов, 34%) на Zerto micro-application для HPE GreenLake Storage Console. Cross-origin ShellAPI bridge. RBAC, 12 feature flags, Keycloak SSO. Accessibility (axe-core, ARIA). Performance: virtualized scrolling, AG Grid infinite row model. CI/CD: Jenkins, Docker, Kubernetes, Helm, AWS ECR. Переход на documentation-first, AI-assisted delivery.
**Что я вынес:** Enterprise governance и процессы. Micro-frontend архитектура. Kubernetes в production. Переход от IC к Tech Lead. AI-assisted development workflow.

## Вывод: кто я

Я — фронтенд-архитектор, который строит визуальные системы для сложных данных. Мой стек — не React/TypeScript (это инструменты). Мой стек — это домены: от consumer tech до enterprise cybersecurity, от 3D-рендеринга до AI-инфраструктуры.

Я умею войти в незнакомую индустрию, понять её entity model, и построить UI, через который люди управляют этой сложностью. Я делал это 12 раз в 10 разных доменах.

Двенадцать компаний. Десять доменов. Два собственных стартапа. Экзиты работодателей на ~$1B+. Путь от SQL Server и Flash до MCP-серверов и AI-агентов. Ни одного формального диплома в CS.

Одна красная нить через всё: **если данные сложные — я делаю их видимыми.**
